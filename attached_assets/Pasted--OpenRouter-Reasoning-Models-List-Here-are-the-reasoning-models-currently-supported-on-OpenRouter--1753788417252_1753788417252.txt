OpenRouter also mentions upcoming support for **Gemini Thinking models**[2], though specific model names weren't detailed in the search results.

## **Key Features for Reasoning Models**

To access reasoning tokens from these models, you need to add `include_reasoning: true` to your API request[2][7]. The reasoning output will appear in the `reasoning` field of each message[2][4].

Example API usage:
```python
payload = {
    "model": "deepseek/deepseek-r1",
    "messages": messages,
    "include_reasoning": true
}
```

These reasoning models provide transparent insight into their thinking process, showing step-by-step reasoning chains that can be useful for complex problem-solving, debugging model logic, and educational purposes[1][2].
